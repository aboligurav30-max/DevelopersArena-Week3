# Week 3 - Sales Data Analysis

## Project Overview

This Python project analyzes a sales dataset using the Pandas library. It loads data from a CSV file, explores the dataset, performs basic data cleaning, calculates important sales metrics, and generates a formatted sales report.

## Objectives

- Learn the basics of data analysis
- Understand the Pandas library
- Read data from CSV files
- Explore datasets
- Handle missing values
- Calculate business metrics
- Generate a simple analysis report

## Features

- Loads sales data from a CSV file
- Displays the first five rows
- Shows dataset information
- Checks rows, columns, and data types
- Identifies missing values
- Removes duplicate records
- Calculates total revenue
- Finds highest and lowest sales
- Calculates average sales
- Identifies the best-selling product
- Displays product-wise revenue

## Technologies Used

- Python 3.x
- Pandas

## How to Run

Open the terminal inside the project folder.

Install the required library:

```bash
pip install -r requirements.txt
```

Run:

```bash
python sales_analysis.py
```
