# Sales Data Analysis Report

## Project Objective

The objective of this project is to analyze a sales dataset using Python and the Pandas library. The project demonstrates loading, exploring, cleaning, and analyzing real-world sales data.

---

## Dataset Information

- Total Records: 100
- Total Columns: 7

Columns:

- Date
- Product
- Quantity
- Price
- Customer_ID
- Region
- Total_Sales

---

## Data Cleaning

The following preprocessing steps were performed:

- Checked for missing values.
- Filled missing Quantity values (if any).
- Removed duplicate records.

Result:

- Missing Values Found: **0**
- Duplicate Records Removed: **0**
